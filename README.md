# vetor
atividade 1
O usuário informa o valor de n, que define o tamanho do vetor.
Em seguida, ele digita os elementos que serão armazenados no vetor X.

Depois disso, é criado um vetor Y, que recebe os mesmos valores de X, porém em ordem inversa (do último para o primeiro).

Ao final, o programa mostra na tela os dois vetores: o original (X) e o invertido (Y).

atividade 2
O programa lê dois vetores, X e Y, ambos com tamanho n.

Em seguida, cria um vetor Z, que pode ter até 2n posições.
Os elementos de X e Y são adicionados em Z, mas sem repetir valores já existentes.

Uma variável chamada k é usada para controlar quantos elementos foram realmente inseridos no vetor Z.
